# Enterprise Network Design Principles

## Objective

Design a secure, scalable, and highly available enterprise network infrastructure.

## Core Principles

- High Availability
- Redundancy
- Security
- Scalability
- Easy Management
- Performance
- Reliability

## Network Layers

- Core Layer
- Distribution Layer
- Access Layer

## Services

- Active Directory
- DNS
- DHCP
- NTP
- Syslog
- Backup
- Monitoring

## Routing

- OSPF
- Static Routing
- BGP (where required)

## Security

- Firewall
- VLAN Segmentation
- VPN
- ACL
- MFA
